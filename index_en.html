Overview of Random Nutrition

Random Nutrition is a mobile app that randomly generates balanced meals while respecting predefined nutritional constraints. The app produces appetizers, main courses, cheeses, and desserts by taking into account energy, protein, fat, carbohydrate, and other macronutrient needs.

It also offers the option to add or exclude certain types of foods to customize meals according to your preferences and requirements.

Main Features

  • Random meal generation that respects nutritional limits for each course (appetizer, main course, cheese, dessert).
  • Customization of nutritional limits (energy, macronutrient ratios).
  • Management of food quantities with predefined ranges.
  • Automatic verification of nutritional constraints for each generated meal.
  • Compliance with dietary guidelines: each meal is balanced and meets daily energy, protein, fat, and carbohydrate requirements.

App Pages

Home Page

Users can view their usage statistics and discover some random foods.

Foods Page

This page displays two lists:

  • The default food list.
  • The customizable food list that users can add to.

By clicking the + button, the user is redirected to a page for creating a custom food entry, which includes the name, categories, nutritional values, a description, etc. Each food item is clickable to display a popup with detailed information.

Meal Generation Page

On this page, the user can:

  • Enter the desired nutritional values.
  • Select the number of meals to generate.
  • Individually configure each meal by selecting the courses and desired food categories.
  • Generate the meals by clicking the "Generate" button once configured.

A loading screen with a tip appears, and once complete, the generated meals are displayed in detail, with the option to save them individually or as a set.

Meals & Meal Sets Page

This page displays two lists:

  • Saved meals.
  • Saved meal sets.

Each item is clickable to open a popup with the details of the meal or set. Only the name and color can be modified.

Settings Page

Users can configure various options in the app to customize their experience.
